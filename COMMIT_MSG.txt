- **Sleep detection guards** — Added 16-hour max gap limit (prevents multi-day non-use from false detection). Fixed condition to accept gaps where either end falls within sleep hours (21:00-10:00), reverting over-strict `bothNearSleep` check that rejected 20:00→08:00 gaps.
- **Fixed `waketime_minutes is not defined`** in `ExternalPage.tsx:1609` — tooltip used undefined variables instead of local `wakeMin`/`appExitMin`.
- **Add Session modal confusion** — Duration picker sat on right side of `→` arrow, looking like an end time field. Added live end time preview (`Session: 17:00 → 19:00`), label changed to `Duration (not end time)`.
- **`manualSessionDate` UTC bug** — `new Date().toISOString().split('T')[0]` gave UTC date. Changed to local date via `getFullYear()/getMonth()/getDate()`. Fixes "Add Session" showing wrong date near midnight.
- **`toLocalDate` in `get-sleep-trends`** — Changed from `iso.split('T')[0]` (UTC date) to `new Date(iso).getFullYear()/getMonth()/getDate()` (local date). Fixes chart bars labeled with UTC date instead of local bedtime date.
- **Chart subtitle** — Updated to "Each bar = one night's sleep. Date = evening you went to bed."
- **Sleep date shift (AM → previous day)** — `getSleepGroupDate` shifts bedtime after midnight to the previous evening's bar. Wake-up date tooltip logic updated: if bedtime is AM (<720min), wake-up always shows `day.date + 1` since the group was shifted.
- **Removed debug AFK button** — Deleted `triggerAfkDebug` callback, ref, `trigger-afk-debug` event listener, `Ctrl+Shift+Alt+A` keyboard shortcut, and 🐛 AFK button from ExternalPage header.
- **AFK popup UI** — Wider (`max-w-sm` → `max-w-xl`), responsive height (`max-h-[min(640px,85vh)]`), edge times row positioned at each divider below timeline bar (shows actual clock times at segment boundaries), segment list taller (`max-h-52` → `max-h-72`), activity picker taller (`max-h-36` → `max-h-48`).
