# 📖 Glossary

**Purpose:** Domain-specific terminology and definitions.

---

## 🌌 Solar System Terms

| Term | Definition |
|------|-----------|
| **Orbit Radius** | Distance from sun to planet (in AU) |
| **Eccentricity** | How elliptical the orbit (0 = circle, 0.9 = very elliptical) |
| **Inclination** | Tilt of orbit relative to ecliptic plane |
| **Longitude of Perihelion** | Angle where planet is closest to sun |
| **Semi-Major Axis** | Average distance from sun |
| **Semi-Latus Rectum** | Parameter for Keplerian orbit calculation |

---

## 💻 Technical Terms

| Term | Definition |
|------|-----------|
| **IPC** | Inter-Process Communication (main ↔ renderer) |
| **Preload** | Script that bridges main and renderer securely |
| **Context Bridge** | Electron API for exposing safe APIs to renderer |
| **R3F** | React Three Fiber (React wrapper for Three.js) |
| **PBR** | Physically Based Rendering |
| **UV Mapping** | How 2D textures wrap around 3D objects |
| **SRGBColorSpace** | Color space for display-referred textures |
| **RepeatWrapping** | Texture repeats across surface |
| **ClampToEdgeWrapping** | Texture edge pixels repeat |

---

## 🗄️ Database Terms

| Term | Definition |
|------|-----------|
| **Better-SQLite3** | Synchronous SQLite wrapper for Node.js |
| **Prepared Statement** | Precompiled SQL query with parameters |
| **JSON Fallback** | Alternative storage when SQLite unavailable |
| **NODE_MODULE_VERSION** | ABI version for native Node.js modules |

---

## 🎨 Visual Terms

| Term | Definition |
|------|-----------|
| **Procedural Texture** | Texture generated algorithmically (not from file) |
| **Normal Map** | Texture that simulates surface bumps |
| **Emissive** | Self-illumination of material |
| **Roughness** | How rough/smooth surface appears |
| **Metalness** | How metallic surface appears |
| **Ambient Light** | Uniform light from all directions |
| **Hemisphere Light** | Sky/ground gradient light |
| **Point Light** | Light from single point source |

---

## 📊 Data Terms

| Term | Definition |
|------|-----------|
| **Activity Log** | Record of app usage (app, duration, timestamp) |
| **Focus Time** | Total time spent on an app |
| **Session** | Continuous usage period of an app |
| **Category** | App classification (IDE, Browser, etc.) |
| **Foreground App** | Currently active window |
| **Background Apps** | Apps running but not in focus |

---

## 🔧 Build Terms

| Term | Definition |
|------|-----------|
| **Renderer** | React UI that runs in Chromium |
| **Main Process** | Node.js process that controls Electron |
| **Vite** | Fast build tool for frontend |
| **CommonJS** | Module system for Node.js (.cjs) |
| **ES Modules** | Modern module system (.mjs/.js) |
| **HashRouter** | Router that uses URL hash (for file://) |

---

## 🐛 Error Terms

| Term | Definition |
|------|-----------|
| **NODE_MODULE_VERSION mismatch** | Native module compiled for wrong Node.js version |
| **ERR_FILE_NOT_FOUND** | Requested file doesn't exist |
| **contextBridge** | Electron secure IPC bridge |
| **contextIsolation** | Security feature separating renderer from Node.js |
