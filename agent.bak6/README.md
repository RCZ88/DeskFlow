## 🚀 How to Use This System

### For All AI Agents:

1. **ALWAYS start here** - Read `agent/agents.md` before any work
2. **Check your type** - Read `agent/qwen.md` if you're Qwen Code
3. **Understand state** - Read `agent/state.md` for current status
4. **Know the context** - Read `agent/context.md` for architecture
5. **Use available skills** - Check `agent/skills.md`
6. **Follow patterns** - Reference `agent/patterns.md`
7. **Respect constraints** - Follow `agent/constraints.md`
8. **Update state** - After completing work, update `agent/state.md`
